API REST de gerenciamento de tarefas (To-Do List) desenvolvida com Spring Boot, com cadastro de usuários, criação de tarefas e base para autenticação via Basic Auth.
Descrição (completa, para o README):
Este projeto é uma API REST de To-Do List construída com Java 21 + Spring Boot, com foco em praticar os fundamentos de backend: modelagem de entidades, persistência com JPA, criação de endpoints e autenticação básica.
A aplicação permite:
•
cadastro de usuários com senha criptografada usando BCrypt;
•
criação de tarefas associadas ao usuário;
•
armazenamento dos dados com Spring Data JPA;
•
execução com banco em memória H2 para desenvolvimento rápido.
Tecnologias principais:
•
Spring Boot (Web e Data JPA)
•
H2 Database
•
Lombok
•
BCrypt
